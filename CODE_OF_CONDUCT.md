# Code of Conduct

본 프로젝트는 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
을 채택합니다. 한국어 번역본은
[여기](https://www.contributor-covenant.org/ko/version/2/1/code_of_conduct/) 에서 볼 수 있습니다.

## 적용 범위

본 행동 강령은 모든 프로젝트 공간에 적용됩니다 — GitHub 저장소(Issue·PR·Discussion·코드),
Discord·Slack·메일링 리스트, 컨퍼런스·밋업 등 프로젝트를 대표하는 모든 자리.

## 신고

행동 강령 위반 사례는 다음 경로로 신고해 주세요:

- **이메일**: conduct@trsim.org (TODO: 실제 도메인 확보 후 갱신)
- **GitHub**: 비공개 보안 보고 채널 (`SECURITY.md` 참조)

신고는 비공개로 처리되며, 신고자의 신원은 보호됩니다.

## 처리

신고 내용은 Core team이 검토하며, [Contributor Covenant 2.1 의 Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines)
에 따라 다음 단계로 대응합니다:

1. **Correction** — 비공개 경고
2. **Warning** — 공식 경고 + 일정 기간 상호작용 제한
3. **Temporary Ban** — 일정 기간 프로젝트 공간 접근 차단
4. **Permanent Ban** — 영구 추방

각 단계의 적용 기준과 절차는 위 링크의 Enforcement Guidelines 원문을 따릅니다.
